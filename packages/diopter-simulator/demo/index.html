Understanding Diopter + Distance

Myopia (negative diopters): Blur increases with viewing distance. A -3D prescription barely affects phone use but severely impacts TV viewing.

Hyperopia (positive diopters): More consistent blur across distances, but still varies based on accommodation ability.

Distance matters: The same prescription creates different user experiences on different devices and viewing distances.
